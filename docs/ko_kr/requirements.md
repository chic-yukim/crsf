# 요구사항
**Languages**: [English](../requirements.md)

## 하드웨어
- OpenGL 4.3 이상 지원하는 그래픽카드

## 소프트웨어
- Windows 10 64bit
- Visual Studio
  - Visual Studio 2013 Update 5: https://www.visualstudio.com/vs/older-downloads/
  - Visual Studio 2017
- CMake: https://cmake.org/download/

### 선택사항
- CUDA 7.5 이상 지원: 네트워크 코덱 라이브러리 사용 시
