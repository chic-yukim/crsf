# Requirements

## Hardware
- GPU supported above OpenGL 4.3

## Software
- Windows 10 64bit
- Visual Studio
  - Visual Studio 2013: https://www.visualstudio.com/vs/older-downloads/
  - Visual Studio 2017
- CMake: https://cmake.org/download/

### Optional
- Above CUDA 7.5: When using network codec library
